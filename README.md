## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve - run in port 8080 
or
node server.js - run in port 5000
```

### Compiles and minifies for production
```
yarn run build
```
### Lints and fixes files
```
yarn run lint
```
